# Contacts book Application

this is a simple contacts book application that allows users to add, view, update, and delete contacts to manage their contacts efficiently.
this application is built using Python script and will be run in the terminal, but in the future it will be built with a GUI... 

## Description

you can use this application to store and manage your contacts. It allows you to add new contacts, view existing contacts, update contact information, and delete contacts you no longer need. The application is designed to be user-friendly and efficient, making it easy to keep track of your contacts.

## Getting Started

### Dependencies

* Python 3.10 or higher
* No external libraries are required
* A terminal or command prompt to run the script

### Installing

* make sure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/) or use a package manager like conda or brew.

### Executing program

* for running the program in terminal

python contacts_book.py
